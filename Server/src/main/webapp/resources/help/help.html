Tdd trainer — как играть?

Cервер доступен сейчас в целях ознакомления. http://codenjoy.com/tdd-trainer/

В чем суть игры?

Надо написать своего бота, отвечащего на вопросы сервера. С каждым новым ответом сервер будет задавать все больше и больше новых вопросов.

Если в какой-то момент логика поломается, ты получишь -1000 очков. Каждые 2 минуты сервер будет у тебя спрашивать ответ, если не успеешь дать его -100, если успеешь и ответишь правильно +100.

Первая задача – написать websocket клиента, который подключится к серверу. Затем заставить сервер услышать твои ответы. Далее нужно отвечать на вопросы сервера, разрабатывая свою логику.
